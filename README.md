# FingerPrinter
Android 6.0 及以上的指纹识别，和Android 9.0以上的生物识别（指纹实现）。

###v1.0 (2020.04.19)   
1、6.0及以上部分使用FingerprintManagerCompat实现；   
2、9.0及以上部分使用BiometricPrompt实现，未涉及其他生物识别操作。
